# Boundless Workout Tracker - Roadmap

## Phase 1: Data Foundation
**Goal**: Set up Convex schema and seed workout program data

- Define schema for workouts, workout logs, and program config
- Create workout data for all 14 days
- Set up program start date configuration

## Phase 2: Today Screen
**Goal**: Build the core "what do I do today?" experience

- Calculate current program day from start date
- Display today's workouts with descriptions
- Show workout status (done/not done)
- One-tap completion toggle

## Phase 3: Progress & History
**Goal**: See the full picture

- Calendar/list view of all 14 days
- Visual progress indicator
- View past workout logs with notes
